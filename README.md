Open Hack U 2025 OSAKA

## 実行方法

1. ターミナルで以下を実行し、コンテナを起動する。

```bash
docker compose up -d
```

2. http://localhost:5173 にアクセスする。

3. コンテナを停止する場合は以下を実行する。

```bash
docker compose down
```

## APIキーの設定

1. `backend/.env` を作成する。

2. [Notion](https://www.notion.so/matsuda-takumi/1b3179e2944180f2829df89096efdf13?pvs=4)に記載したAPIキーをコピペする。
